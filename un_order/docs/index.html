Các thẻ HTML cơ bản cho Web

HTML Headings: Dùng để thể hiện tiêu đề, các nội dung muốn nhấn mạnh

HTML Paragraphs: Định dạng 1 đoạn văn bản

HTML Links: Dùng đề liên kết page này với page khác

HTML Images: Để chèn hình ảnh

HTML Tables: Định dạng bảng

HTML Lists: Hiện danh sách

Thuộc tính các thẻ HTML

Alt: Tên hình ảnh, sẽ hiện ra khi bị lỗi đường dẫn ảnh, giúp người xem biết nội dung ảnh đó là gì khi ảnh không thể thể hiện

Disabled: Dùng khi muốn khóa một thẻ nhập liệu trong form (chúng ta sẽ nói kỹ về form trong các bài sau)

Href: link

Id: tên của một đối tượng, sẽ được dùng nhiều khi học CSS và Javascript

Src: Đường dẫn đến file hình, âm thanh (thẻ img)

Style: Khai báo một số giá trị style đối tượng (Sẽ học nhiều khi học css)

Value: Giá  trị mặc định trong 1 form nhập liệu

Vài điều về Style HTML

HTML Styling: định dạng trang html chung

HTML Text Color: định dạng màu sắc chữ

HTML Fonts: Định dạng font chữ

HTML Text Size: kích thước chữ

HTML Text Alignment: Vị trí chữ

Tất cả những style này chúng ta sẽ gặp lại trong phần học CSS

Bố cục của một page

Bố cục 1 trang Web sẽ gồm 3 phần chính

    • Banner phía trên

    • Menu trên (Dưới banner)

    • Menu bên (có thể bên trái, bên phải hoặc cả 2 bên)

    • Footer: Chân trang, dưới cùng trang Web để thể hiện các thông tin như địa chỉ, liên lạc, web liên kết,…

    • Và phần giữa, lớn nhất, đạp vào mắt người dùng đầu tiên là nội dung Web
